# COVID-19 Time Series Analysis (Python)

## Overview
This project analyzes global COVID-19 time-series data to understand trends in confirmed cases and daily growth.

## Dataset
- Source: Kaggle (COVID-19 Day-wise Data)
- Data Type: Time series

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Steps
- Data validation (nulls, duplicates)
- Date-based sorting
- Feature creation (daily new cases)
- Trend visualization

## Key Insights
- Global confirmed cases show exponential growth over time
- Daily new cases highlight peak and decline phases of the pandemic

## Files
- `data/` – Dataset
- `notebook/` – Analysis notebook
